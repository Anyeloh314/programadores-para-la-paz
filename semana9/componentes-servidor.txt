Sistema operativo:
Es el software principal del equipo y administra los recursos del hardware. Permite ejecutar aplicaciones y servicios como Node.js.
Node.js:
Es un entorno de ejecución que permite ejecutar JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.
Aplicación server.js:
Es el archivo donde se encuentra el código del servidor. Define las rutas y las respuestas que enviará la aplicación.
Puerto:
Es un punto de comunicación que utiliza el servidor para recibir solicitudes. En esta actividad se utiliza el puerto 3000.
Logs:
Son los mensajes que genera el servidor para informar sobre su funcionamiento, errores o eventos importantes.
